DeepStructureAI é uma plataforma multiagente para pesquisa científica assistida por IA.

Ele combina:

- Research Team
- Consensus Engine
- Research Sessions
- Knowledge Graph
- Scientific Memory
- Knowledge Manager
- Obsidian
- Model Router
- Model Registry
- Ollama / Hermes
- QuantumTool
