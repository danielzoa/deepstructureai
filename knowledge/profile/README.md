# Perfil pessoal

Este diretório contém apenas contexto pessoal revisável pelo usuário.

- `identity.json`: identidade, papéis e interesses.
- `preferences.json`: idioma e preferências de colaboração.
- `boundaries.json`: limites de uso e privacidade.

Não inclua senhas, tokens, documentos, dados financeiros ou outros segredos.
O DeepStructureAI carrega somente os três arquivos JSON listados acima.
